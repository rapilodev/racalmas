calcms - manage broadcasts at your community radio station.

calcms consists of two components:

  • edit the broadcast schedule
  • publish the broadcast schedule

    • calcms itself is stand-alone, but can be integrated into both static and dynamic web sites.

    Documentation can be found here:

  • http://dev.radiopiloten.de/dokuwiki/doku.php/calcms:start
